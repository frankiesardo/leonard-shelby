<summary>What is a "desired-state system" for organizations and how does it remove politics?</summary>

You define organizational principles as code (team sizes, reporting ratios, title levels, compensation rules), compute what the company should look like using a solver, and then reconcile the delta against reality. When someone requests 50 new hires, the system shows what else changes -- budgets shift, other teams shrink. Decisions become simulations, not negotiations. HR's job becomes finding the minimum steps from current state to desired state.
