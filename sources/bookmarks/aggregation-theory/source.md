# Aggregation Theory

*Source: https://stratechery.com/2015/aggregation-theory/*
*Note: Full text behind Stratechery paywall*

Ben Thompson's foundational 2015 essay introducing Aggregation Theory. The Internet made distribution free, which made supply abundant. When supply is abundant, power shifts to whoever controls demand — the user relationship. Aggregators (Google, Facebook, Amazon, Uber) win by owning the consumer relationship and commoditizing suppliers. Zero marginal costs, zero distribution costs, and a virtuous cycle: more users attract more suppliers attract more users. The result is winner-take-all dynamics fundamentally different from pre-Internet monopolies.
